# Potato Disease Classification

A Deep Learning-based image classification system that detects potato leaf diseases from images using Convolutional Neural Networks (CNNs).

## Overview

This project classifies potato leaf images into the following categories:

- Early Blight
- Late Blight
- Healthy

The objective is to enable early disease detection and support farmers in reducing crop losses through automated image analysis.

## Features

- Automated potato leaf disease detection
- Image classification using Deep Learning
- Data preprocessing and augmentation
- Model training and evaluation
- Disease prediction with confidence scores

## Tech Stack

- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset

The project uses the PlantVillage Potato Leaf Dataset.

| Class | Description |
|---------|-------------|
| Potato___Early_blight | Early Blight Disease |
| Potato___Late_blight | Late Blight Disease |
| Potato___healthy | Healthy Potato Leaf |

